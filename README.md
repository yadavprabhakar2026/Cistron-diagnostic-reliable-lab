# Cistron Diagnostic Reliable Lab Website

Premium React and Node.js website for Cistron Diagnostic Reliable Lab in Biratnagar, Nepal.

## Run Locally

```powershell
npm.cmd install
npm.cmd run dev
```

Backend:

```powershell
Copy-Item .env.example .env
npm.cmd run server
```

## Included

- React, Vite, Tailwind CSS, React Router, Framer Motion, React Icons
- Home, About, Mission & Vision, Services, Blog, Gallery, Contact, and Admin pages
- Contact booking form with success message and focused Book a Test flow
- SEO meta handling, Open Graph tags, Schema.org structured data, robots.txt, sitemap.xml
- Express API with MongoDB models, contact submissions, Nodemailer email hook, JWT protected admin routes
